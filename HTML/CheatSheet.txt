HTML Cheat Sheet
1. Basic Structure
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>


2. HTML Elements
	Headings: <h1> to <h6>
	Paragraphs: <p>
	Links: <a href="URL">Link Text</a>
	Images: <img src="image.jpg" alt="Description" />
	Lists:
		Ordered List: <ol><li>Item 1</li></ol>
		Unordered List: <ul><li>Item 1</li></ul>
	Tables:
<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

3. Forms
•	Form Structure:
<form action="submit_form.php" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <input type="submit" value="Submit">
</form>

•	Input Types: text, password, email, checkbox, radio, file, submit

4. Semantic HTML
•	Articles and Sections:
<article>
    <h2>Article Title</h2>
    <p>Content of the article.</p>
</article>

<section>
    <h2>Section Title</h2>
    <p>Content of the section.</p>
</section>
•	Navigation: <nav><a href="home.html">Home</a></nav>
•	Footer and Header: <footer>Footer content</footer>, <header>Header content</header>

5. Multimedia
•	Audio:
<audio controls>
    <source src="audio.mp3" type="audio/mp3">
    Your browser does not support the audio element.
</audio>

•	Video:
<video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>


6. HTML5 Features
•	Canvas:
<canvas id="myCanvas" width="200" height="100"></canvas>
<script>
    var canvas = document.getElementById('myCanvas');
    var ctx = canvas.getContext('2d');
    ctx.fillStyle = '#FF0000';
    ctx.fillRect(0, 0, 150, 75);
</script>

•	Local Storage:
<script>
    localStorage.setItem('key', 'value');
    var value = localStorage.getItem('key');
</script>


7. Accessibility
•	ARIA Roles:
		Role: role="button", role="navigation", etc.
•	Alt Text for Images:
		<img src="logo.png" alt="Company Logo">


8. Meta Tags
	Character Set: <meta charset="UTF-8">
	Viewport: <meta name="viewport" content="width=device-width, initial-scale=1.0">
	Description: <meta name="description" content="Page description here">
	Keywords: <meta name="keywords" content="HTML, CSS, JavaScript">


9. HTML Entities
•	Common Entities:
	&lt; for <
	&gt; for >
	&amp; for &
	&quot; for "


10. Forms Advanced
•	Fieldset and Legend:
<fieldset>
    <legend>Personal Information</legend>
    <!-- Form elements here -->
</fieldset>

•	Datalist:
<input list="browsers" name="browser" id="browser">
<datalist id="browsers">
    <option value="Chrome">
    <option value="Firefox">
    <option value="Safari">
    <option value="Edge">
</datalist>
